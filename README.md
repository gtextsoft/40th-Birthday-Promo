## üîß Customization Guide

### Update Contact Information
1. Open `index.html`
2. Find the WhatsApp links (search for "YOUR_WHATSAPP_NUMBER")
3. Replace with actual WhatsApp number (e.g., 2348012345678)
4. Update footer contact details

### Update Estate Images
- Replace images in `assest/` folder
- Keep filenames: `jasper.jpeg`, `garnet.jpeg`, `sapphire.jpeg`, `sadiurs.jpeg`

### Modify Colors
- Open `styles.css`
- Edit CSS variables in `:root` section
- Main colors: `--primary-navy`, `--accent-gold`, `--celebration-purple`

### Add Analytics
- Insert Google Analytics or Facebook Pixel code in `<head>` section of `index.html`
